## Missing Programs

### Chancenkarte (Opportunity Card) - NOT IN SYSTEM

The Opportunity Card (Chancenkarte) launched in June 2024 is a significant new program not currently in the system. It allows skilled workers to enter Germany for job seeking based on a points system.

**Recommendation:** Add Chancenkarte program to Germany offerings.
